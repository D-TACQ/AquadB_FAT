# AquadB_FAT

## Support for AquadB FAT

```bash
DATA/  :: empty directory, your capture data will show up here
DWG/wiggle.gz  :: sample STIM pattern for one ENCODER. be sure to gunzip this in place.
LICENSE
PCFG/* :: configuration files for analysing, plotting data 
PYEPICS/*    :: uut configuration files, typically deployed with and run on the uut
```
